# ToDo Capstone Project (HTML + CSS + JavaScript)

A clean, responsive To-Do web app built with vanilla HTML/CSS/JS.
Includes Add/Submit, Search, Delete, Mark Done, and LocalStorage persistence.

## Features
- Add a new task (Submit)
- Live search / filter
- Delete tasks
- Mark task as done/undone
- Clear completed tasks
- Clear all tasks
- Persists tasks using LocalStorage (data stays after refresh)
- Responsive UI (mobile + desktop)

## Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- Nginx (for container hosting)
- Docker
- Jenkins (CI/CD)
